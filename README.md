# Craft Attack 13 Community Bot

## Installation

1. Stelle sicher, dass du [Python](https://www.python.org/) auf deinem Computer installiert hast (empfohlen: Version 3.6 oder höher).
2. Klone dieses Repository oder lade es als ZIP-Datei herunter.
3. Installiere die erforderlichen Abhängigkeiten mithilfe des folgenden Befehls:

   ```sh
   pip install -r requirements.txt

# Konfiguration

1. Erstelle eine Anwendung auf der Discord Developer-Seite.
2. Generiere einen Bot-Token und kopiere ihn.
3. Erstelle eine Datei namens .env im Hauptverzeichnis des Projekts.
4. Füge den folgenden Inhalt in die .env-Datei ein und setze den Bot-Token ein:

   ```sh
   TOKEN=DEIN_BOT_TOKEN_HIER_EINFÜGEN

# Verwendung

1. Führe den Bot aus, indem du den Befehl ausführst:

   ```sh
   python main.py

2. Der Bot sollte sich jetzt mit deinem Discord-Server verbinden. Du kannst Befehle verwenden, um den Bot zu steuern und die Funktionen zu nutzen.
